Script em Python que identifica o signo do usuário a partir da data de nascimento e toca uma música aleatória de uma playlist local.

Este foi um dos meus primeiros projetos em Python, desenvolvido enquanto eu aprendia programação de forma autodidata.

🚀 Funcionalidades: 

*Identificação de Signo*
O usuário insere a data de nascimento no formato DD/MM, e o programa identifica automaticamente o signo correspondente utilizando estruturas condicionais (if e else).

*Música Aleatória*
Após identificar o signo, o programa seleciona aleatoriamente uma música da pasta de arquivos .mp3.

*Controle de Repetição*
O script registra as músicas já tocadas em um arquivo (tocadas.txt), evitando repetições até que todas as músicas da pasta tenham sido reproduzidas.

*Validação de Entrada*
O programa valida o formato da data inserida para evitar erros de execução.

🛠️ Tecnologias utilizadas:

Python

Biblioteca random

Biblioteca playsound

▶️ Como executar

Instale a dependência necessária:

pip install playsound

Crie uma pasta chamada musicas e coloque seus arquivos .mp3 nela.

Execute o script Python.

Digite sua data de nascimento no formato:

DD/MM

O programa exibirá seu signo e tocará uma música aleatória da playlist.
